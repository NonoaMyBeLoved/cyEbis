# divideTab

노래 악기 사운드 조절해서 타브 악보 따는 용으로 만듦.

구글링하면 나오는 사이트들 돈 내야 노래 풀로 변환 가능하던데(아님 말고), 괘씸해서 직접 코덱스로 짬.

기타 타브 귀카피는 좀 빡쌜거 같고, 베이스는 하기 편할 거임.

타브 악보도 추출하도록 하고 싶었는데 잘 안돼서 일단 비활성화 상태.

GPU 없어도 CPU만으로 되긴 하던데, 느리고 사양이 딸리면 안될 수 있음.

(라이젠7 9700X, RTX 5060 Ti 사용 중인데 GPU 사용하면 얼마 안 걸리고, CPU는 좀 걸렸음. 자동으로 하면 CPU만 사용하는 거 같음.)

소리가 깔끔하지 않을 수 있는데, 신디사이저 부분이 좀 이상해서 얘를 유동적으로 조절하면 좋습니다.

실행하는 법은 윈도우 검색에서 `cmd` 검색하면 명령 프롬프트라고 나오는데, 이거 실행 후 `backend` 폴더로 들어가서 아래 명령어를 복붙하면 됩니다.

```powershell
.\.venv\Scripts\python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

실행되면 브라우저에서 아래 주소로 들어가면 됩니다.

```text
http://127.0.0.1:8000
```
